node环境搭建

  • 官网上下载与自己操作系统相对应的node安装包
  • 安装只需要一路next就好了
  • 完成后打开命令行窗口输入:node -v
  • 成功输出版本号就代表搭建成功

创建一个简单的服务器


			// 加载http模块
			// http是node内置模块，所有可以直接加载使用
			var http = require("http");
			
			
			// 创建一个服务器
			var server = http.createServer(function(req,res){
				res.end("This is a simpile server.")
			})
			
			// 设置服务器端口为3000，启动服务
			server.listen(3000)
			
			
			/*
				步骤:
					1.创建index.js,内容如上
					2.打开命令行窗口,输入node index
					3.打开浏览器，输入：localhost:3000
					4.查看页面上是否显示 This is a simpile server.
					5.显示代表成功,不显示请重新检查是否有什么遗漏的
			*/
